Garry's Mod Setup Instructions (Windows)

To Play
  1. Download the Counter-Strike: Source Textures
    • Unzip the folder and drop it into addons: C:\Program Files (x86)\Steam\SteamApps\common\GarrysMod\garrysmod\addons
  2. Login and Subscribe to the EPMR collection on Steam (you must be friends with LiquidMotion)

To Host (after completing above steps)

Run this script to automatically do all of the following steps

    Extract the gma file
  1. Navigate to C:\Program Files (x86)\Steam\steamapps\common\GarrysMod\bin and find gmad.exe
  2. In another window, navigate to C:\Program Files (x86)\Steam\steamapps\common\GarrysMod\garrysmod\addons.
    • Drag custom_roles_for_ttt_1215502383.gma on top of the gmad.exe file name.
    • This will create a custom_roles_for_ttt_1215502383 folder that is also inside of your addons folder.

    Use the right configuration
  3. Now download init.lua and config.cfg (right click > save link as)
    • Replace the init.lua file at C:\Program Files (x86)\Steam\steamapps\common\GarrysMod\garrysmod\addons\custom_roles_for_ttt_1215502383\gamemodes\terrortown\gamemode
    • Replace the config.cfg file at C:\Program Files (x86)\Steam\steamapps\common\GarrysMod\garrysmod\cfg
